Slot filling

Basic slot filling

This feature helps you easily get needed information from users. One way to get information is to ask the user little by little. The second way is that the user can type the information at once. All you need to do is have information in the entity and you can solve both situations easily.

Example 2: Ask for information about size

Slot filling with one parameter

  1. Create an entity with information.

  1. Create intent linked to the entity.

  1. Prepare interactions: create interactions for small/middle/big tree offers.

  1. Add intent to interactions.

  1. Add entity to interactions.

  1. Set value of entity (depends on the size of trees in interaction).

  1. Create interaction which do not get the size of a tree. small/middle/big tree offers.

  1. Add intent and entity to interaction.

  1. Set value of entity to optional.

  1. Add also handlers for responses..
